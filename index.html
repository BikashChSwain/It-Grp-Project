Inheritance in Java

Inheritance is an important piller of OOP(Object - Oriented Programming). It is the mechanism in java by which one class is allowed to inherit the features(field and methods) of another class.

Sub-Topics To Be Discuss


All the below subtopics will cover under the topic Inheritance In Java.

  • Basics of inheritance.
  • Types of inheritance.
  • Method overriding.
  • Dynamic method dispatch.
>

Why use inheritance in java?

For Method Overriding (so runtime polymorphism can be achieved).
For Code Reusability.

Types of inheritance with proper examples.

Single, Multiple, Multilevel, Hierarchical & Hybrid.

Rules for implement method overriding.

The method must have the same name as in the super-class and must be an IS-A relationship.
