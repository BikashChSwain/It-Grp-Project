Types Of Inheritance

A class can inherit the properties and methods from various classes like one superclass for subclass or more than one subclass for a superclass or nested subclasses for a superclass . Java provides various types of inheritance such as

  • Single Inheritance
  • Multilevel Inheritance
  • Hierarchical Inheritance

    • Java does not support multiple inheritance . Multiple inheritance leads to ambiguous problems . Instead of multiple inheritance Java uses interfaces to implement multiple inheritance.

    Single Inheritance

    The properties and methods of a base class are derived by a subclass and this is called single inheritance . To derive the data members and methods of the baseclass , a subclass uses extends keyword in the class declaration statement. The flow diagram shows that class B extends only one class which is A. Here A is a parent class of B and B would be  a child class of A.

    The general form of single inheritance is:

    Class baseclass
{
	……………………..
	……………………..
}
Class subclass extends baseclass
{
	……………………..
	……………………..

}

    Example:

    class Animal
{  
 	void eat()
	{
		System.out.println("eating");
	}  
}  
class Dog extends Animal
{  
 	void bark()
	{
		System.out.println("barking");
	}  
}  
class TestInheritance
{  
 	public static void main(String args[])
	{  
    	Dog d=new Dog();  
    	d.bark();  
   		d.eat();  
    }
}  

    
Output:
barking
eating

    Multilevel Inheritance

    Here the base class is derived by the subclass and the derived class inherited by another subclass one after another . It is a ladder or hierarchy of single level inheritance . Multiple classes are involved in inheritance but one class extends only one class . The lowermost subclass can make use of all its superclass members . As you can see in below flow diagram C is subclass or child class of B and B is a child class of A.

    The general form of multilevel inheritance is:

    Class baseclass
{
	……………………..
}
Class subclass1 extends baseclass
{
	……………………..
}
Class subclass2 extends subclass1
{
	……………………..
}

    Example:

    class Animal
{  
	void eat()
	{
		System.out.println("eating");
	}  
}  
class Dog extends Animal
{  
	void bark()
	{
		System.out.println("barking");
	}  
}  
class BabyDog extends Dog
{  
	void sit()
	{
		System.out.println("sit");
	}  
}  
class TestInheritance2
{  
	public static void main(String args[])
	{  
		BabyDog d=new BabyDog();  
		d.sit();  
		d.bark();  
		d.eat();  
	}
} 

    
Output:
sit
barking
eating

    Hierarchical Inheritance

    HeThe superclass properties and methods can be inherited by multiple subclasses. That means many subclasses can derive the properties of one base class . Class B,C and D inherits the same class A. A is parent class (or base class) of B,C & D.

    The general form of hierarchical inheritance is:

    Class baseclass
{
	……………………..
}
Class subclass1 extends baseclass
{
	……………………..
}
Class subclass2 extends baseclass
{
	……………………..
}
Class subclass3 extends baseclass
{
	……………………..
}

    Example:

    class Animal
{  
    void eat()
	{
		System.out.println("eating");
	}  
}  
class Dog extends Animal
{  
    void bark()
	{
		System.out.println("barking");
	}  
}  
class Cat extends Animal
{  
    void meow()
	{
		System.out.println("meowing");
	}  
}  
class TestInheritance3
{  
    public static void main(String args[])
	{  
   		 Cat c=new Cat();  
   		 c.meow();  
   		 c.eat(); 
    }
}  

    
Output:
meowing
eating

    Multiple Inheritance (Through interface)

    In Multiple inheritances, one class can have more than one superclass and inherit features from all parent classes. Java does not support multiple inheritances with classes. Multiple inheritance is not supported by Java using classes , handling the complexity that causes due to multiple inheritance is very complex. It creates problem during various operations like casting, constructor chaining etc. In java, we can achieve multiple inheritances only through Interfaces. Class C is derived from interface A and B.

    Example:

    interface AnimalEat 
{
   void eat();
}
interface AnimalTravel 
{
   void travel();
}
class Animal implements AnimalEat, AnimalTravel 
{
   	public void eat()
 	{
      System.out.println("Animal is eating");
  	}
   	public void travel()
 	{
      System.out.println("Animal is travelling");
   	}
}
public class Demo
{
   	public static void main(String args[]) 
	{
      Animal a = new Animal();
      a.eat();
   	  a.travel();
   	}
}


    
Output:
Animal is eating
Animal is travelling

    Hybrid Inheritance (Through Interface)

    It is a mix of two or more of the above types of inheritance. Since java doesn’t support multiple inheritances with classes, hybrid inheritance is also not possible with classes. In java, one can achieve hybrid inheritance only through Interfaces.

    Example:

    public class ClassA 
{
    public void dispA()
    {
        System.out.println("disp() method of ClassA");
    }
}
public interface InterfaceB 
{
    public void show();
}
public interface InterfaceC 
{
    public void show();
}
public class ClassD extends ClassA implements InterfaceB,InterfaceC
{
    public void show()
    {
        System.out.println("show() method implementation");
    }
    public void dispD()
    {
        System.out.println("disp() method of ClassD");
    }
    public static void main(String args[])
    {
        ClassD d = new ClassD();
        d.dispD();
        d.show();
    }
}



    
Output:
disp() method of ClassD
show() method implementation
